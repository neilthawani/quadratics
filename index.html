Projectile Motion

You might see many examples of quadratic equations "in the wild." Here, we find ourselves as accidental bird-watchers. These particular birds are limited, though, by an overpowering desire to evolve into slingshot fodder. Experiment with the slingshot below and see how you can make these birds fly. It looks like these birds are only able to fly if they aim high!
  1. When user stops holding onto bird-ball, send bird along bezier curve to landing point. Rotate the color of the ball as it's flying through the air.
  2. Send slingshot rope back to initial point.
  3. Go back and try to make it so the bird walks or hops into the slingshot. You can do this with a bezier animation.
  4. Send scaffold message when ball lands. Give student the option to reset the game or move on. Roll credits if they decide to move on.
