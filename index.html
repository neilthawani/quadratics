Projectile Motion

You might see many examples of quadratic equations "in the wild." Here, we find ourselves as accidental bird-watchers. These particular birds are limited, though, by an overpowering desire to evolve into slingshot fodder. Experiment with the slingshot below and see how far you can make these birds fly. Pull back on slingshot. Dynamically draw line between x,y point on slingshot and mouse point. Draw dotted-line curve from mouse to estimated landing point. When user stops holding onto bird-ball, send bird along bezier curve to landing point.
